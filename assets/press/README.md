# Press Kit / EPK

Carpeta reservada para cuando quieras dejar disponible un press kit descargable (PDF con bio,
foto de prensa en alta resolución, logo, y links) — por ejemplo `assets/press/zolan-epk.pdf`.

Si lo agregas, puedes enlazarlo desde la sección "Conecta" en `index.html` agregando una tarjeta
más dentro de `.conecta__grid`:

```html
<a class="conecta__card" href="assets/press/zolan-epk.pdf" target="_blank">
  <span class="conecta__card-label">Descargar EPK</span>
  <span class="conecta__card-arrow">↓</span>
</a>
```
